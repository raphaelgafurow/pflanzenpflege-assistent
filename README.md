# Pflanzenpflege-Assistent (Desktop-Anwendung)

**Autor:** Raphael Gafurow  
**Projektart:** Schulprojekt (Desktop)  
**Technologie:** Avalonia mit .NET

## Projektbeschreibung

Der Pflanzenpflege-Assistent ist eine Desktop-Anwendung zur Verwaltung von Zimmerpflanzen und deren Pflege.
Ziel ist es, Pflanzen strukturiert zu erfassen und den Überblick über anstehende Pflegearbeiten wie Gießen oder Düngen zu behalten.

Dies ist keine Mobile-App, sondern eine PC/Desktop-Anwendung.

## Funktionen

### Pflanzenverwaltung

- Pflanzen anlegen
- Pflanzen löschen
- Übersicht aller Pflanzen

### Pflege-Regeln

- Pflege-Regeln hinzufügen
- Pflege-Regeln ändern
- Pflege-Regeln löschen
- Berechnung und Anzeige der nächsten fälligen Pflege

### Pflege-Einträge

- Pflege-Einträge hinzufügen
- Pflege-Einträge ändern
- Pflege-Einträge löschen
- Historie pro Pflanze anzeigen

### Suche

- Filter in der Übersicht nach Name, Art oder Standort
